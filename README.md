## Overview
Simple full-stack e-commerce cart with:
- React + Vite frontend (Tailwind CSS)
- Express backend + SQLite DB
- Auth (register/login/logout), cart CRUD, checkout (mock receipt)
- Toast messages via react-toastify
- Bonus: Fake Store API sync endpoint
